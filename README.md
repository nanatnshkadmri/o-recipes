# o-recipes

## Introduction
This project is part of The Odin Project Foundations course. The goal is to build a basic recipe website using only HTML.

## Skills Demonstrated
By completing this project, I will have practiced:

- Creating HTML pages
- Using HTML boilerplate
- Working with headings, paragraphs, and lists
- Adding links between pages
- Inserting images
- Organizing files and folders
- Using Git and GitHub for version control
